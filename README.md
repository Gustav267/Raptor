# Raptor - Regressionsanalyse potentiometrischer Titrationen fuer optimierte Resultate

## Installation
1. Installiere Python 3.14 von hier: [https://www.python.org/downloads/windows/](https://www.python.org/downloads/windows/) oder mit winget in PowerShell:

```powershell
winget install -s winget Python.Python.3.14
```

2. Starte die PowerShell neu und verifiziere, dass Python 3.14 installiert ist:

```powershell
python --version
```

Der Output dieses Befehls sollte so oder so ähnlich aussehen:

```
Python 3.14.6
```

3. Kopiere den foldenden Text und füge ihn in die PowerShell ein (Rechte Maustaste fügt Text ein):

```powershell
irm https://raw.githubusercontent.com/Gustav267/Raptor/main/InstallRAPTOR.ps1 | iex
```
